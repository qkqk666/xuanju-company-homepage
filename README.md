# 玄驹科技主页

企业官方网站，基于 HTML + CSS + JavaScript 单文件部署，托管于 Cloudflare Pages。

## 技术栈
- 前端：HTML + CSS + JavaScript（单文件）
- 动画：GSAP + ScrollTrigger + Lenis + Three.js
- 部署：Cloudflare Pages
- 版本管理：GitHub Desktop

## 本地预览
直接用浏览器打开 `index.html` 即可（file:// 协议）。

## 部署
推送到 GitHub 后，Cloudflare Pages 自动构建并发布。
